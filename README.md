Branch Walletの公式ホームページです。サービスのコンセプト、特徴、使い方をわかりやすく説明しています。

## 📁 ファイル構成

```
website/
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptアニメーション
└── README.md       # このファイル
```

## 🚀 ローカルでの確認方法

### 方法1: シンプルなHTMLファイルとして開く
```bash
# websiteディレクトリで
open index.html
```

### 方法2: ローカルサーバーを起動
```bash
# websiteディレクトリで
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

### 方法3: Live Serverを使用（VSCode拡張機能）
1. VSCodeでindex.htmlを開く
2. 右クリック → "Open with Live Server"

## 📝 カスタマイズ方法

### 色を変更
`styles.css`の`:root`セクションでカラー変数を変更

### コンテンツを変更
`index.html`の各セクションのテキストを編集

### レイアウトを変更
`styles.css`でグリッドレイアウトやpadding/marginを調整

## 🚢 デプロイ

### Vercel
```bash
cd website
vercel --prod
```

### Netlify
```bash
cd website
netlify deploy --prod --dir=.
```

### GitHub Pages
1. GitHubリポジトリにpush
2. Settings → Pages → Source: main/website
